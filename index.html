metodología

En este especial empleamos técnicas de extracción de datos a través de las API de Spotify, YouTube y Billboard. El análisis cuantitativo se realizó a través de scripts configurados en Python y librerías de acceso libre. También validamos la data mediante análisis descriptivo estadístico y en el caso de Spotify recurrimos a la normalización y clusterización de variables tomando como referencia el modelo de David Rudkin. En el análisis cualitativo, como la valoración de sentimiento de la discografía de Shakira, se empleó la librería Transformers de Hugging Face.
